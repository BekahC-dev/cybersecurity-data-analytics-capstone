# Improving Cybersecurity Decision-Making Using Data Analytics

This project analyzes the CICIDS2017 dataset to explore how data analytics and simple machine learning models can improve cybersecurity alert prioritization.

## Overview
The goal of this project is to determine whether basic models can help identify potential attacks more effectively than a simple baseline approach.

## Dataset
- CICIDS2017 (MachineLearningCSV)
- Includes both imbalanced (Tuesday) and more balanced (Friday) data

## Methods
- Data cleaning and preprocessing
- Exploratory data analysis
- Baseline comparison
- Logistic Regression
- Decision Tree

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- False Positive Rate

## How to Run
Open the notebook in Google Colab and run all cells from top to bottom.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
